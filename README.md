## Минимальные системные требования:  
* Linux, macOS, WSL  
* PHP >= 8.3  
* Composer - глобальная версия  
* Make  
* Git  

## Установка  
Клонируем проект  
```
git clone git@github.com:pavlovAlex96/php-project-45.git

```  
Заходим в директорию игры  
```
cd php-project-45
```  
Устанавливаем необходимые для работы игры библиотеки  
```
make install
```  
## Запуск игр  
*Для запуска доступны 5 игры по следующим командам:*  
* Проверка на чётность - ```make brain-even```  
* Калькулятор - ```make brain-calc```  
* Наибольший общий делитель (НОД) - ```brain-gcd```  
* Арифметическая прогрессия - ```brain-progression```  
* Простое ли число? - ```brain-prime```
